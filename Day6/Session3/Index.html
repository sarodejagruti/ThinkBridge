🌐 Web Development Keywords

A simple guide to HTML, CSS, and JavaScript

HTML

HTML Annotated Example

<!DOCTYPE html>  This line represents that following document is HTML type
<html lang="en"> HTML open tag with attribute lang="en"
<head>  head opening tag
  <meta charset="UTF-8"> character set of Unicode UTF-8
  <meta name="viewport" content="width=device-width, initial-scale=1.0"> viewport width based on 
 device, initial zoom 100%
   <title>Document</title> title of website, appears on tab
</head> head closing tag
<body> body opening tag
  Hello World
</body> body closing tag
</html> HTML closing tag

Metadata & SEO

          
         <meta> → Defines metadata like charset, viewport, description, keywords.
         <link> → Links external resources (CSS, favicon, etc.).
         <style> → Embeds CSS inside HTML
         <script> → Adds JavaScript.
         <base> → Sets a base URL for relative links.

Formatting Tags

      <b> → Bold text (stylistic).
      <strong> → Important text (semantic bold).
      <i> → Italic text (stylistic).
      <em> → Emphasized text (semantic italic).
      <u> → Underlined text.
      <mark> → Highlighted text.
      <small> → Smaller text.
      <del> → Deleted (strikethrough).
      <ins> → Inserted (underline).
      <sup> → Superscript (e.g., 10²).
      <sub> → Subscript (e.g., H₂O).
      <code> → Inline code.
      <kbd> → Keyboard input.
      <samp> → Sample output.
      <abbr> → Abbreviation (with tooltip).
      <cite> → Citation for work.
      <q> → Inline quotation.
      <blockquote> → Long quotation.

Lists

      <ul> → Unordered list (bullets).
      <ol> → Ordered list (numbers).
      <li> → List item.
      <dl> → Definition list.
      <dt> → Definition term.
      <dd> → Definition description.

Links & Navigation

      <a> → Hyperlink.
      <nav> → Section for navigation links.

Images & Media

      <img> → Displays image.
      <picture> → Multiple image sources for responsive design.
      <figure> → Container for images/diagrams.
      <figcaption> → Caption for <figure>.
      <audio> → Embed audio.
      <video> → Embed video.
      <source> → Media source (used inside <audio>, <video>, <picture>).
      <track> → Subtitles/captions for media.
      <iframe> → Embed another page.

Tables

      <table> → Defines a table.
      <tr> → Table row.
      <td> → Table cell (data).
      <th> → Table header cell.
      <thead> → Header section.
      <tbody> → Body section.
      <tfoot> → Footer section.
      <caption> → Table caption.
      <colgroup> → Group of columns.
      <col> → Column formatting.

Forms & Input

      <form> → Defines a form.
      <input> → Input field.
      <textarea> → Multi-line text input.
      <button> → Button.
      <select> → Drop-down list.
      <option> → Item in a drop-down.
      <optgroup> → Group inside <select>.
      <label> → Label for form control.
      <fieldset> → Group of form fields.
      <legend> → Caption for <fieldset>.
      <datalist> → List of predefined input values.
      <output> → Output result.
      <progress> → Progress bar.
      <meter> → Scalar measurement (like temperature, rating).

Semantic & Layout

      <header> → Page/section header.
      <footer> → Page/section footer.
      <main> → Main content.
      <section> → Thematic section.
      <article> → Self-contained content (like blog post).
      <aside> → Sidebar content.
      <address> → Contact information.

Interactive & Scripting

      <details> → Expandable/collapsible content.
      <summary> → Title for <details>.
      <dialog> → Dialog box / popup.
      <canvas> → Drawing graphics via JavaScript.
      <noscript> → Fallback content if JS disabled.
      <template> → Hidden content for reuse.
      <slot> → Placeholder inside web components.
      <shadow> → Shadow DOM root (deprecated).

Obsolete / Deprecated

      <font> → Font styling (use CSS instead).
      <center> → Center content (use CSS instead).
      <big> → Bigger text (use CSS instead).
      <marquee> → Scrolling text (not recommended).

CSS

Text & Font

      color → Text color.
      font-family → Font type (e.g., Arial, Verdana).
      font-size → Size of the text.
      font-weight → Thickness (normal, bold, 100–900).
      font-style → Italic, normal, oblique.
      text-align → Align text (left, center, right, justify).
      text-decoration → Underline, line-through, none.
      text-transform → Uppercase, lowercase, capitalize.
      letter-spacing → Space between letters.
      word-spacing → Space between words.
      line-height → Line spacing.
      white-space → How spaces/wraps are handled.
      direction → Text direction (ltr, rtl).
      vertical-align → Align text vertically in line.

Backgrounds

      background-color → Background color.
      background-image → Add an image as background.
      background-repeat → Repeat, no-repeat, repeat-x/y.
      background-position → Position of background.
      background-size → Size (cover, contain, px, %).
      background-attachment → Scroll or fixed background.
      background-clip → Painting area (border-box, content-box).
      background-origin → Positioning area.
      background-blend-mode → How background layers blend.

Box Model

      width / height → Element size.
      max-width / max-height → Maximum size.
      min-width / min-height → Minimum size.
      margin → Outside spacing.
      padding → Inside spacing.
      border → Border around element.
      border-radius → Rounded corners.
      box-sizing → Defines how width/height are calculated.
      box-shadow → Adds shadow.

Positioning & Display

      position → static, relative, absolute, fixed, sticky.
      top, right, bottom, left → Position offsets.
      z-index → Stacking order.
      display → block, inline, flex, grid, none.
      visibility → visible, hidden, collapse.
      overflow → visible, hidden, scroll, auto.
      clip / clip-path → Define visible area.
      float → Align left or right.
      clear → Control floating elements.

Flexbox

      display: flex → Activates flexbox.
      flex-direction → Row, column, row-reverse, column-reverse.
      flex-wrap → Wrap items or not.
      justify-content → Align items horizontally.
      align-items → Align items vertically.
      align-content → Align rows.
      flex → Shorthand for grow, shrink, basis.
      flex-grow → Item’s growth factor.
      flex-shrink → Item’s shrink factor.
      flex-basis → Default size of an item.
      order → Item order.

CSS Grid

      display: grid → Activates grid.
      grid-template-columns / grid-template-rows → Define rows & columns.
      grid-column-gap / grid-row-gap → Gap between items.
      grid-gap → Shorthand for row + column gap.
      grid-template-areas → Name areas in a grid.
      grid-area → Assign item to area.
      grid-column / grid-row → Span across lines.
      justify-items → Horizontal alignment inside grid cells.
      align-items → Vertical alignment inside grid cells.
      justify-content / align-content → Align whole grid.

Colors & Effects

      opacity → Transparency (0 to 1).
      filter → Effects like blur, brightness, grayscale.
      mix-blend-mode → Blending mode with background.
      cursor → Mouse cursor style.

Transitions & Animations

      transition → Shorthand for transition effects.
      transition-property → Which property animates.
      transition-duration → Animation time.
      transition-timing-function → Easing type.
      transition-delay → Start delay.
      animation-name → Keyframe animation name.
      animation-duration → Length of animation.
      animation-iteration-count → Number of times.
      animation-direction → Normal, reverse, alternate.
      animation-fill-mode → State before/after animation.
      animation-timing-function → Easing curve.

Transformations

      transform → rotate, scale, skew, translate.
      transform-origin → Transformation origin point.
      perspective → 3D perspective.

Lists & Tables

      list-style-type → Bullet/number style.
      list-style-image → Use image for bullets.
      list-style-position → Inside or outside bullets.
      border-collapse → Collapse or separate table borders.
      border-spacing → Space between table cells.
      caption-side → Caption position.
      empty-cells → Show/hide empty cells.
      table-layout → Auto or fixed layout.

Media & Responsive

      @media → Media queries for responsiveness.
      object-fit → How images/videos fit container.
      object-position → Image/video position inside container.

Miscellaneous

      content → Insert content (with ::before, ::after).
      quotes → Custom quotation marks.
      resize → Make element resizable.
      outline → Outline around element.
      outline-offset → Space between outline & element.
      clip-path → Shape clipping.

JavaScript

JavaScript is a programming language that makes web pages interactive. It can update content, validate forms, and handle user events.
