HTML Annotated Example

  <meta charset="UTF-8"> character set of Unicode UTF-8

   <title>Document</title> title of website, appears on tab
</head> head closing tag
<body> body opening tag
  Hello World
</body> body closing tag
</html> HTML closing tag
